## How to run this code

You must create and activate the environment from the YAML file. From the root of the directory, run

```bash
conda env create -f environment.yml
conda activate nlp-assign1
```

The investigation conducted for the trigram and byte pair encoding is done in the Python Notebooks: /notebooks/trigram.ipynb and /notebooks/bpe.ipynb. In order to run these notebooks, launch Jupyter using the command

```bash
jupyter lab
```

And proceed to navigate to the notebooks folder, and run the entire notebook with the "restart and run all" button.
